Ch 12 & 13 - JBoss Clustering

All build targets create two application server configurations named node1 and node2. After the build, to run the application servers, open two command windows, cd to the applications server's bin directory in each, and enter the following in one window:

./run.sh -c node1 (Linux)
run -c node1 (Windows)

Then enter the following in the other:

./run.sh -c node2 (Linux)
run -c node2 (Windows)

None of the targets creates a database.

Target: 01
Section: 12.2
Description: Creates a two node cluster and deploys an EJB load balancing example to it.
After the build:

After building and starting both servers, run the 01-client target to see a counter load balance across the two servers.

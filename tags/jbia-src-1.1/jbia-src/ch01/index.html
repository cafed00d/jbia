Ch 1 - Vote for JBoss

All build targets create an application server configuration named voteforjboss. After the build, to run the application server, open another command window, cd to the applications server's bin directory, and enter:

./run.sh -c voteforjboss (Linux)
run -c voteforjboss (Windows)

None of the targets creates a database.
